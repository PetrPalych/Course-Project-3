## Run Locally

Clone the project

```bash
  git clone https://github.com/petrPalych/Course-Project-3
```

Go to the project directory

```bash
  cd .../.../Course-Project-3
```

Install dependencies

```bash
  all need files in jar package
```


To start program

```bash
  You can start program from Main.class in IDE
  or 
  use "java Main" in terminal
```
